Coding Conventions

  • Should there be a space after the return operator ^ ?
  • Should there be spaces inside the square brackets of a block?
  • Should there be spaces around the vertical bar (|) that separates parameters from code in a block?
  • Should there be spaces around @ when creating a Point?
  • Should the keywords of long keyword messages be on separate lines, indented from the receiver on the first line?
  • Should the last statement in a method or block be terminated with a period even though it isn't required?
  • Should there be a black like after the line that declares local variables?
