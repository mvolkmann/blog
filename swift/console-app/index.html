Overview

The Console app is a macOS app that can be used to view log messages.

Steps to Use

To view log messages from an iOS app running on an iPhone:

  1. Open the Finder.
  2. Navigate to the Applications folder.
  3. Navigate to the Utilities subfolder.
  4. Double-click Console.app.
  5. Select the phone from the devices list in the left nav.
  6. Click the start button at the top to begin capturing log messages.
  7. Click "All Messages" at the top of the message area.
  8. Run the iOS app.
  9. By default only Error and Fault messages are displayed. To see other messages, click the "Action" menu and select "Include Info Messages" and "Include Debug Messages".
  10. Select the subsystem whose messages should be displayed. IT IS NOT CLEAR HOW TO DO THIS.
