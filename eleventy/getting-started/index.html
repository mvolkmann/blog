Getting Started

Here are the most basic steps to create and build an 11ty site.

  1. Create a new directory and cd to it.

  2. Enter npm init -y to create a package.json file.

  3. Enter npm install -D @11ty/eleventy.

  4. Create the directory _includes.

  5. Add the file layout.md in the _includes directory containing the following. The meta tags are necessary to get a perfect Lighthouse score.

    <!DOCTYPE html>
<html lang="en">
  <head>
    <title>My Site</title>
    <meta charset="utf-8" />
    <meta name="description" content="My Site" />
    <meta name="viewport" content="width=device-width, initial-scale=1" />
  </head>
  <body>
    {{content | safe}}
  </body>
</html>

  6. Create the file index.md file containing the following:

    ---
layout: layout.md
---

# Welcome to my site!

  7. Enter npx eleventy --serve

  8. Browse localhost:8080 to see the site.

This uses BrowserSync to provide "hot reload". It watches the current directory for file changes. When they are detected, it rebuilds the site and refreshes the browser. Note that having a <body> element is required for hot reload to work. The layout file above provides this.

If the project is placed in a Git repository, add _site/ to the .gitignore file since that only contains generated code.

Optionally add these npm scripts in package.json that can be used to build and serve the site:

"build": "eleventy",
"clean": "rm -rf _site",
"start": "eleventy --serve"
