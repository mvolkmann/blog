Overview

Spring is a set of Java services that aim to simplify some common tasks. Spring Boot "makes it easy to create stand-alone, production-grade Spring based Applications".

Steps

  1. Install the latest version of the Java Development Kit (JDK).

  2. Install the Gradle build tool. In macOS, this can be done with brew install gradle.

  3. Open a terminal.

  4. cd to the directory where a new project will be created.

  5. Create a directory for a new project and cd to it.

  6. gradle init.

  7. For the type of build to generate, select "1. Application".

  8. For the implementation language, select "1. Java".

  9. For the target Java version, press enter to accept the default version of Java.

  10. For the project name, press enter to accept the default project name which matches the directory name.

  11. For the application structure, select "1. Single appplication project".

  12. For the build script DSL, select "1. Kotlin".

  13. For the test framework, select "1. JUnit 4".

  14. For "Generate build using new APIs and behavior", enter "yes".

  15. Open the project in your editor of choice.

  16. Open the file app/src/main/java/org/example/App.java.

  17. Add the following lines before the class definition.

    import org.springframework.boot.autoconfigure.SpringBootApplication;

    @SpringBootApplication
