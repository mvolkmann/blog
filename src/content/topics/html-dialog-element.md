The `<dialog>` element was added to HTML in version 5.2.
It provides a way to define dialogs that are initially closed.
These can be styled with CSS.

See my Object Computing (OCI) article [Dialogs with Style: The New HTML &lt;dialog&gt; Element](<https://objectcomputing.com/resources/publications/sett/may-2019-html-dialog-element?v=1.1.1>).
