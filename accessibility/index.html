Top Mistakes

96% of all accessibility errors fall into the following six categories.

Low Contrast Text

The contrast between text background and foreground colors should be at least 4.5:1.

Missing Labels

All form control elements such as input, textarea, and select should have an associated label element.

Empty Links

Anchor (a) elements whose content is an image rather than text should specify a concise name using the aria-label or aria-labelledby attribute.

Empty Buttons

button elements, and elements that act as buttons, whose content is an image rather than text should specify a concise name using the aria-label or aria-labelledby attribute.
