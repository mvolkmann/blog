Extracting Images

One way to extract images from a PDF file is to use the browser-based tool PDFCandy. The free tier has a limit of one download per hour.

To use this:

  1. Browse PDFCandy.
  2. Click the "PDF to JPG" button.
  3. Click the "+ ADD FILE" button.
  4. In the dialog that appears, select a PDF file and click the "OK" button.
  5. Choose a quality from the dropdown such as "High Quality".
  6. Click the "Convert PDF" button.
  7. Wait for the task to complete. This can take around one minute.
  8. Click the "Download File" button which downloads a ZIP file containing one JPG file for each image found in the PDF.
  9. Unzip the downloaded ZIP file to access the images.
