Svelte Material UI

Svelte Material UI is a library of Svelte components that implement Material UI. For a list of supported components, click the link above and see the list in the left nav.

To install it in a Svelte project:

  1. cd to the project root directory.

  2. npm install svelte-material-ui

  3. npm install @smui/common

  4. Copy node_modules/svelte-material-ui/bare.css to public/svelte-material-ui.css.

  5. Add the following line in public/index.html after the last <link> tag:

    <link rel="stylesheet" href="/svelte-material-ui.css" />

Here's an example of using the Button component in a Svelte component.

<script>
  import Button from '@smui/button';

  let clicked = 0;
</script>

<main>
  <Button on:click={() => clicked++}>Click Me</Button>
  <p>clicked = {clicked}</p>
</main>
